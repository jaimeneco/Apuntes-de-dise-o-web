Positions

Static

Posición normal o por defecto: no se aplica ninguna propiedad de posición.

Static

Relative

Afecta la posición Top-right-bottom-left, y conserva su espacio.

Relative

Absolute

Afecta a la posición Top-right-bottom-left, pero no ocupa su espacio orginal. Queda flotando entre los otros elementos. Se posiciona con respecto al padre más cercano que tenga una propiedad position diferente de static, normalmente será al relative. Si no la encuentra utiliza el body.

Absolute

Fixed

Afecta la pisition Top-right-bottom-left. No ocupa espacio, se posiciona respecto al navegador, se mantiene fijo n la pantalla (no se mueve con el scroll)

Fixed

Sticky

Comportamiento combinado entre static y fixed. Utiliza el position top y left como punto de adhesión. Se posiciona respecto a la ventana del navegador(viewport)y se queda fijo en la pantalla hasta que el elemento padre se acaba.
